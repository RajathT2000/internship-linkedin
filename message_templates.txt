## Key Points to Remember:
✓ Always personalize with their first name
✓ Mention Sydney location (important for local opportunities)
✓ Highlight MQ PACE verification (shows legitimacy)
✓ Keep it brief and respectful
✓ Show genuine interest in their company
✓ Mention specific skills (RAG agents, SQL automation)
✓ Be polite and professional
✓ Don't sound desperate - be confident but humble

## DO NOT:
✗ Use generic copy-paste messages
✗ Be too pushy or demanding
✗ Send multiple messages to the same person
✗ Mention automation tools
✗ Make spelling or grammar mistakes
✗ Be overly casual or use emojis
✗ Write long paragraphs (keep it concise)
